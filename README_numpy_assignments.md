# 🔢 NumPy Assignments — Beginner to Intermediate

A collection of 4 NumPy practice assignments completed in **Google Colab**, covering everything from basic array creation to indexing, broadcasting, sorting, and stacking. Built as part of my Python & Data Science learning journey. 🌱

---

## 📁 Repository Structure

```
numpy-assignments/
│
├── 01_array_creation.ipynb               # Assignment 1 — Array creation basics
├── 02_indexing_and_slicing.ipynb         # Assignment 2 — Attributes, indexing & slicing
├── 03_operations_and_broadcasting.ipynb  # Assignment 3 — Operations & aggregates
├── 04_fancy_indexing_and_sorting.ipynb   # Assignment 4 — Fancy indexing, sorting & stacking
└── README.md
```

> 📝 Rename your notebook files to match the structure above (or update it to match your actual file names) before uploading.

---

## 📚 Assignment Overview

| # | Assignment | Topics Covered |
|---|---|---|
| 1 | **Array Creation** | `np.array()`, `zeros()`, `ones()`, `full()`, `arange()`, `linspace()`, `eye()`, random arrays |
| 2 | **Attributes, Indexing & Slicing** | `.shape`, `.ndim`, `.size`, `.dtype`, 1D/2D indexing, slicing, boolean indexing |
| 3 | **Operations & Broadcasting** | Element-wise arithmetic, scalar broadcasting, `.sum()`, `.mean()`, `.min()`, `.max()`, axis-wise aggregation |
| 4 | **Fancy Indexing & Sorting** | Fancy indexing (1D & 2D), conditional selection, `np.sort()`, `np.argsort()`, `concatenate()`, `vstack()`, `hstack()` |
